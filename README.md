# Low Level Playground

### Goals:
1. Learn to be productive in a low level language.
2. Get comfortable with manual memory management.
3. Gain a deeper understanding of how computers work.

### Mini Projects:

1. Implement common data structures in C
    - X stack
    - X list
    - X hashmap

2. ARM Assembly
    - X Fibonacci sequence generator
    - X Hash function
    - X List sorter

3. ARM Emulator
    - Emulate a subset of ARM assembly using C
    - Rewrite assembly programs from above using emulator


### How to run

Run C code:
```
source dev.sh
test
```
